What is Anime JS?

Anime.js is a library that can be used in many ways, but used to do motion animation using Javascript. Anime.JS uses an object defined animation solution to animate the changes specified, in our below example, we see a square that is a gradient so we can show with ease, rotation, scale and translation, this is achieved using the library.

We can now start working with Anime JS, to test this we can try creating a simple animate object in our Javascript file, at this point we dont need to specify targets or any parameters, as long as this presents no issues, we can continue

anime({
//targets : *a list of objects we wish to use*,
//*A List of possible transformations we can do*
});

If this does not cause any errors, we can no look at what we can do with Anime JS, First, there are many properties we can change. We can change almost any property we wish, first you make your object, then specify the change you wish to make, like so:

anime({
targets: ".classObject #idObject",
opacity: '0.5',
backgroundColor: '#4e148c',
width: '200px'
});

We can also manipulate things according to built in transforms, it animates change according to css properties like width, height and angles.

anime({
targets: ".classObjec #idObject",
translateX : 100,
scale : 1.5;
});

We can Define keyframes, or rather a timeline, by default the keyframes are the same length dependant on the duration. This is a simple way for the smooth animations on the page, we define keyframes then reset the values on the last one so its a smooth transition.

anime({
targets: ".classObjec #idObject",
keyframes: [
{ translateY: -10, translateX: 10, scale: 0.6, backgroundColor: "#fff" },
{ translateY: 0, translateX: 0, scale: 1, backgroundColor: "#FF0000" },
],
});

There are Many different units that can be used interchangably with anime js, in some of the examples below we see that i dont specify px or a scale in px, this is because they are unitless deffinitions, by default when it comes to translations and width/height, its in px, you can define a unit by adding quotes around the definition and adding the unit on the end. Anime JS also uses relative units IE for scale, by default its a percentage, 1 being 100%, therefore 1.5 is 150%. All movement is relative to the 0, being its starting location. Therefore -100px is definable as back 100px from the start.

anime({
targets: ".classObjec #idObject",
keyframes: [
{ translateY: -10, translateX: '10px'//units, scale: 0.6, backgroundColor: "#fff" },
{ translateY: 0, translateX: 0 // unitless, scale: 1, backgroundColor: "#FF0000" },
],
});

There are also timeline capabilities, you can define a timeline as follows, adding children to it adds objects that will be put on the timeline, they will be manipulated per the timeline and it will happen linearly, IE one after the other, the duration of the timeline is the duration of one animtions, divide accordingly, IE 5000ms /6 will give us the needed time.



var timeline = anime.timeline({
duration: 833.3333333333333,//5000/6
loop: true
});
timeline.add({
targets: '#example41',
translateX: 100
}).add({v targets: '#example42',
translateX: 100
}).add({
targets: '#example43',
translateX: 100
}).add({
targets: '#example41',
translateX: 0
}).add({
targets: '#example42',
translateX: 0
}).add({
targets: '#example43',
translateX: 0
});

Examples

In this example, when the button is clicked the ball will move side to side by 350px, this is achieved using a css selector that targets the class of the object and anime js, the object and usage look like so :

anime({
targets: ".square1",
keyframes: [
{ translateX: -350 },
{ translateX: 0 },
],
duration: 2000,
});

In this example, when the button is clicked the ball will move side to side by 350px, this is achieved using anime.js to targer the id of the object we wish to move and targetting that id using a dom selector. It is achieved like so :

let jsSelector = document.getElementById("square2");
anime({
targets: jsSelector,
keyframes: [
{ translateX: -350 },
{ translateX: 0 },
],
duration: 2000,
});

In this example, when the button is clicked the square will rotate and move. a dom selector. It is achieved like so :

anime({
targets: "#square3",
keyframes: [
{ translateX: -350, rotate: 270 },
{ translateX: 0, rotate: 0 },
],
duration: 5000,
});
